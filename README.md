本题正解：

```c
#define MAX(a, b)   (((a) > (b)) ? (a) : (b))

int MaxSubArray(const int nums[], int numSize)
{
    int sum = 0, res = nums[0];
    for(int i = 0; i < numSize; i++)
    {
        const int n = nums[i];
        sum = MAX(sum + n, n);
        res = MAX(res, sum);
    }
    
    return res;
}

```

## <a name="tube_test_with_mice"></a> 用三只老鼠测八根试管中的一根毒试管的问题

有 **8** 根试管，其中 **只有一根** 试管是有毒的。现在有 **3** 只小白鼠。我们可以通过让小白鼠尝试饮用某一根或某几根试管中混合出来的试剂来测试哪一根试管有毒。现在要求：每只小白鼠只能饮用一次试剂，那如何让这些小白鼠 **同时** 饮用安排好的试剂来 **一次性** 测试出哪根试管有毒？

乍看之下，这个问题似乎比较简单。如果最后不要求 “**同时一次性**” 测试的话，我们最容易想到的是使用折半搜索法来解决。比如，先对8根试管分成前四根与后四根，然后让第一只老鼠先品尝前4根试管混合出来的试剂。如果第一只老鼠没死，那么说明这4根没有毒，而另外4根有毒。随后把推测有毒的4根试管再分成2组，然后让第二只老鼠品尝，前2根试管混合出来的试剂。如果老鼠没死，那说明这两根试剂都没毒，有毒的在另外两根之中；如果死了则说明这两根试剂有其中一根是有毒的。轮到第3只老鼠之后就能判别哪根试剂有毒了——如果死亡，则说明喝到有毒的；若没死，则说明另外一根有毒。

而问题要求是让这3只小白鼠同时品尝试剂，且只能喝一次，一次性得出最终结果。因此我们不能直接用折半搜索算法去解决。而这里有一个显式的关键条件，即——只有一根试剂是有毒的，也就意味着目标对象个数是1。还有一个隐藏的关键条件就是——老鼠品尝试剂后的状态是2值的，“生”或“死”，其中之一。
